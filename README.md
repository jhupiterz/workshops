## Practice material: Jupyter Notebooks

Jupyter Notebooks (`.ipynb` format) are interactive notebooks in which you can write plain text but also write and execute code. 
The best way to open a `.ipynb` file is either:
* in your code editor if you have one (e.g., PyCharm, VSCode, Atom, etc) or
* online on [JupyterLab]([url](https://jupyter.org/try-jupyter/lab/)) (the native online environment for Jupter Notebooks)

If you go for JupyterLab, then you'll have to upload the `.ipynb` files into the workspace.
